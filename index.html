Objetivo de la Materia:

El alumno desarrollará las habilidades necesarias en el uso del lenguaje PHP y MySQL, que le permitan analizar, diseñar y crear sitios web dinámicos que den solución a problemas reales, utilizando herramientas adecuadas para asegurar que cumplan con los requerimientos y estándares de seguridad. Una vez creados, los publicará y administrará.

Competencias a las que contribuye:

  1. Determinar requerimientos de Información de la organización mediante instrumentos o técnicas de recopilación de información para obtener especificaciones técnicas de la base de datos a diseñar.
  2. Proponer diseño de base de datos mediante diagramas de bases datos que satisfagan los requerimientos de la organización.
  3. Implementar bases de datos apegándose a la propuesta de diseño para satisfacer las necesidades de información de la organización.
  4. Mantener la operación de la base de datos apegándose a las políticas de seguridad y bajo los estándares reconocidos por la industria para su correcto desempeño.
  5. Determinar los requerimientos de información utilizando instrumentos de recolección de datos identificando políticas y necesidades de la organización para desarrollar soluciones de software.
  6. Documentar los requerimientos de Información utilizando técnicas para su clasificación, representación y codificación para establecer un registro estandarizado de requerimientos.
  7. Modelar el sistema de información con base en la documentación de requerimientos para evaluar y mejorar el desempeño de los procesos actuales.
  8. Proponer el sistema de información de acuerdo a la evaluación del modelo y a la disponibilidad de recursos para satisfacer las necesidades de la organización.
  9. Formular planes de desarrollo de sistemas con base en las estimaciones de tiempo y recursos para alcanzar los objetivos acordados.
  10. Ejecutar los planes de desarrollo de sistemas estableciendo mecanismos de control para verificar el cumplimiento de los objetivos de la organización.
  11. Seleccionar las métricas de desempeño de acuerdo a los estándares reconocidos que aseguren la correcta operación del sistema de información.
  12. Evaluar el desempeño del sistema de información de acuerdo a las métricas establecidas para verificar su correcta operación.

Temas:

  1. Introducción a las aplicaciones WEB
  2. Bases de datos.
  3. Diseño físico relacional
  4. PHP
  5. Elementos Avanzados de Programación
  6. Seguridad en Aplicaciones Web
